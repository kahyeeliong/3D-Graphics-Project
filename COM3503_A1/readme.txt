Author:
Name: Liong Kah Yee
Email: kyliong1@sheffield.ac.uk

Overview
This project is a 3D graphics application created using Modern OpenGL. It features a spacecraft room environment with hierarchical models, animations, lighting, and interactivity.

The key elements of the project include:

1. A hierarchical model of two animated robots.
2. A globe with rotation and lighting effects.
3. A spacecraft room environment with textured walls and floors.
4. Interactive and animated spotlight attached to Robot 2.
5. Dynamic lighting and material effects for realistic rendering.
